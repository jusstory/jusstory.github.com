백엔드 개발자들에게 요구되는 프로그래밍 언어 핵심 8가지 Skill set

  • HTML (HyperText Markup Language)

    흔히 마크업으로 많이 불리며, 웹 페이지를 구성하는 골격 과 같아요. 간단히 말하자면 웹 페이지 제작의 기본 언어로 제목, 단락, 목록 등과 같은 본문을 위한 구조적 의미를 나타냅니다. 또 링크, 인용과 그 밖의 항목으로 구조적 문서를 만들 수 있는 방법을 제공해요. 웹 퍼블리싱에서 가장 기본이 되는 툴이며 웹 접근성과 SEO(검색 최적화)에 많은 영향을 줍니다.

  • CSS (Cascading Style Sheets)

    마크업 언어가 실제 표시되는 방법을 기술하는 언어로 웹 페이지의 비주얼을 담당합니다. 즉 HTML은 웹 사이트의 몸체이며 CSS는 옷과 액세서리처럼 꾸미는 역할을 담당해요. HTML과 XHTML에 주로 쓰이며, XML에서도 사용됩니다.

  • JavaScript

    동적 표현을 하기 위해 필요한 언어(객체 기반의 스크립트 프로그래밍 언어)로 정적이었던 웹페이지를 동적으로 작동시킵니다. 예를 들면 클릭 후 페이지가 넘어가는 것과 회원 가입 페이지에서 정보가 올바르게 입력되지 않았을 시 알림이 뜨는 상황을 생각하시면 됩니다.

  • HTML5

    웹 문서를 제작하는 데 쓰이는 기본 프로그래밍 언어 HTML(hypertext markup language)의 최신규격으로, 웹페이지를 구성하는 데 있어 지배적인 역할을 하는 마크업 언어(markup language)에요. 별도 설치 없이 웹 브라우저 상에서 다양한 멀티미디어 콘텐츠와 앱 등을 바로 제공할 수 있습니다.

  • JAVA

    객체지향프로그래밍 언어로 C/ C++에 비해 간략하고 쉬운 편이에요. 또한 네트워크 분산처리 환경에서 사용하기 위해 만들어진 언어로, 바이러스가 침투하지 못하는 구조를 지니고 있습니다. 따라서 보안에 매우 강하며 자바 가상 머신(JVM, Java Virtual Machine)을 사용해 운영 체제와는 독립적으로 동작 시킬 수 있어요. 자동 메모리 관리 등을 지원해 다른 언어에 비해 안정성이 높은 것도 장점입니다.

  • Python

    오픈소스 고급 프로그래밍 언어 중 하나로 초보자뿐만 아니라 전문가들도 포함하는 매우 다양한 사용자 층을 가지고 있는 프로그램입니다. 플랫폼이 독립적이어서 다양한 플랫폼에서 쓸 수 있고 기본으로 제공되는 라이브 러리가 많은 것이 특징입니다. 라이브러리(모듈)이 풍부한 만큼 손쉬운 사용, 빠른 개발속도가 큰 장점으로 꼽힙니다.

  • PHP

    하이퍼텍스트 생성 언어에 포함되어 동작하는 스크립팅 언어. 별도의 실행 파일을 만들 필요 없이 HTML 문서 안에 직접 포함시켜 사용할 수 있으며, C, 자바, 펄 언어 등에서 많은 문장 형식을 준용하고 있어 동적인 웹 문서를 빠르고 쉽게 작성할 수 있습니다.

  • LINUX

    1990년대 초 핀란드 대학생 리누스 토발즈(Linus B. Torvalds)가 구역 내 근거리통신망(LAN)이나 웹사이트에 사용됐던 유닉스(UNIX)를 대체하기 위하여 개발한 컴퓨터 운영체제입니다. 안정성과 신뢰도를 높이는 운영체제로 인정 받으면서 인터넷에 소스가 공개된 무료 프로그램 이어서 누구나 사용할 수 있고 컴퓨터 전문가가 언제든 개선할 수 있다는 것이 장점입니다.

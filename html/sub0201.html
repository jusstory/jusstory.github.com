Skill

html

  • div 구조 구축은 물론 header, section, article, footer 등의 구조적 요소 사용 가능 audio, video 등 멀티미디어적 요소 사용 가능 EditPlus, Atom 등의 에디터 사용.

css

  • html 구조들을 원하는 위치에 배치하고 폰트 설정, 그라디언트, 다중 배경 지정 등 디자인에 따른 꾸미기 가능. 애니메이션 사용으로 동적 꾸밈 가능.

jQuery

  • 특수 효과 및 애니메이션 구현 가능. 배너 슬라이딩 기능, 네비게이션 쪽 효과, 모바일 네비게이션 작동 효과 등 홈페이지의 기본 효과는 물론, 페럴렉스 스크롤링, 접혔다 열리는 아코디언 기능, 사진 갤러리, 모달 팝업 구현 가능.

media query(반응형)

  • media Query를 사용하여 데스크탑뿐만 아니라 태블릿PC, 모바일 등 여러 크기의 디바이스 환경에 따라 동적인 디자인 변형 가능. 필요시 jQuery도 사용하여 구현.

웹표준, 웹접근성

  • validator.w3.org를 사용하여 웹표준 점검, k-wah04 또는 openwax를 사용하여 웹접근성 점검하여 100% 맞춤 가능.

CrossBrowsing

  • Chrome, Explorer, Sapari, Opera, Firefox 같은 다양한 브라우저간의 호환성을 맞춰 동일하게 표현 및 최적화 가능. 조건주석문, 브라주저별 css 핵 사용

Plugin, Bootstrap

  • 각종 Plugin, bootsrtap을 사용한 화면 구현

Photoshop, Illustrator, InDesign

  • 홈페이지 디자인 가능. (제안서 홈페이지 디자인 경험有), 아이콘, 이미지 제작 등 홈페이지 유지보수에 필요한 모든 작업 가능. 아이콘 제작 가능, 배너 제작 가능, 편집디자인 경력 有

Word, Excel, PPT 등의 문서작업
